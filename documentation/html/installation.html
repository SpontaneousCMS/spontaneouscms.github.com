Installation Guide¶

This guide will walk you through installing Spontaneous and the software that it requires.

Spontaneous is a Ruby program so first we’ll need to Installing Ruby.

Installing Ruby¶

Install RVM:

$ rvm install 1.9.3

Installing Spontaneous¶

$ gem install spontaneous

This installs the Spontaneous library files (and their dependencies) as well as the executable spot.

Installing a Database¶

Spontaneous is tested & works with MySQL 5+ and PostgreSQL 9+ and is currently completely agnostic about the choice.

This may change in a later version however with the way that PostgreSQL is developing (with native JSON and hash support).

On MacOS X we recommend you use homebrew to install either of these.

$ brew install postgresql
# or
$ brew install mysql
